# react-express
Todo List with server part and Da. 

## Actions(All will be saved in database): 
<ul>
  <li>Add todo</li>
  <li>Remove todo</li>
  <li>Mark as done</li>
</ul>

## Technologies: 
<ul>
  <ol>react</ol>
  <ol>redux</ol>
  <ol>nodejs</ol>
  <ol>express</ol>
  <ol>mongoDB</ol>
  <ol>redux-saga</ol>
  <ol>reselect</ol>
  <ol>redux-actions</ol>
</ul>
</br>

## Run
```js
npm run server 
cd client
npm start
```
